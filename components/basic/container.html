# Container 布局容器

用于布局的容器组件，方便快速搭建页面的基本结构：

<xn-container>：外层容器。当子元素中包含 <xn-header> 或 <xn-footer> 时，全部子元素会垂直上下排列，否则会水平左右排列。

<xn-header>：顶栏容器。

<xn-aside>：侧边栏容器。

<xn-main>：主要区域容器。

<xn-footer>：底栏容器。

以上组件采用了 flex 布局，使用前请确定目标浏览器是否兼容。此外，<xn-container> 的子元素只能是后四者，后四者的父元素也只能是 <xn-container>
