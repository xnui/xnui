# DatePicker 日期选择器

用于选择或输入日期

# 选择日

以「日」为基本单位，基础的日期选择控件

基本单位由 type 属性指定。

<xn-date-picker v-model="value"></xn-date-picker>

__ 显示代码

# 选择日期范围

可在一个选择器中便捷地选择一个时间范围

在选择日期范围时，默认情况下左右面板会联动。

<xn-date-range-picker v-model="value"></xn-date-range-picker>
